# Data-analyysi - Opintotehtävä 1
Koodi analysoi palomuurin dataa kuukaudelta. Repo sisältää raakadatan puhtaaksi jalostamisen, ip-osoitteiden tietojen hakemisen ja datan visualisoinnin. Koodi on kirjoitettu Pythonilla ja käyttää useita kirjastoja, kuten pandas, matplotlib ja seaborn.

# Ajaminen
Ajaaksesi koodin uudelleen nimeä .env-esimerkki .env:ksi ja täytä tarvittavat API-avaimet. Sitten voit ajaa koodin Python-ympäristössä.
